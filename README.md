## JavaScript and UI Components Resource Hub

Welcome to this open-source repository designed to serve as a practical learning resource for students, educators, and developers looking to master frontend web development. This repository contains a curated collection of standalone mini-projects and modular UI components built entirely with vanilla technologies.

The core objective of this resource is to help learners build a strong structural understanding of foundational web engineering before moving on to abstract modern frameworks like React.

### Learning and Educational Objectives

This repository provides clear, real-world examples to help students grasp:
- DOM Architecture: How to select, traverse, and dynamically manipulate HTML elements using vanilla JavaScript.
- State Management: Implementation of native logic and CSS classes to track and update UI states, such as light and dark modes.
- Event Handling: How to capture and manage user interactions cleanly, efficiently, and without performance bottlenecks.
- Clean Code Practices: Demonstration of modular, semantic, and human-readable code architecture that adheres to industry standards.
